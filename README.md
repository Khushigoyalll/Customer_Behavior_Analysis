
# 👨🏻‍💻Customer Behavior Data Analyst Portfolio Project

Customer Shopping Behavior Analysis is an end-to-end Data Analytics project that explores customer purchasing patterns and business performance using Python, SQL, and Power BI. The project focuses on data cleaning, exploratory analysis, customer segmentation, revenue analysis, and dashboard development to identify trends and support data-driven decision-making.

The final outcome includes business insights, interactive visualizations, and actionable recommendations aimed at improving customer engagement, loyalty, and overall business performance.




## 📂 Dataset Overview

The dataset contains customer shopping transactions and behavioral information.
####  Dataset Summary


|  Metric    | Value               |
| :--------  | :------------------------- |
| `Total Records` | 3,900|
| `Total Columns`  | 18 |
| `Missing Values`| 37 |
| ` Domain ` | Retail / Consumer Behavior  |


## ⚙️Tools & Technologies Used

- Microsoft Excel (Dataset Source)
- Python (Pandas)
- Jupyter Notebook
- PostgreSQL
- Power BI
- Gamma (Presentation Creation)

## 🚀 Project Workflow

- Loaded the dataset in Jupyter Notebook.
- Performed data cleaning and preprocessing.
- Conducted Exploratory Data Analysis (EDA) using Python.
- Executed SQL queries in PostgreSQL to extract business insights.
- Built an interactive dashboard in Power BI.
- Generated actionable insights from customer behavior data.


## 📊 Dashboard Features

- 👥 Customer Segmentation Analysis
- 🔔 Analyzes customer subscription status and trends.
- 📊 Tracks sales performance across different product categories.
- 💰 Revenue Analysis by Gender and Age Group
- ⭐ Top-Rated Product Analysis
- 🎟️ Discount Impact Analysis
- 🚚 Shipping Preference Analysis
- 📈 Interactive Power BI Dashboard
- 📑 Business Reporting and Recommendations
